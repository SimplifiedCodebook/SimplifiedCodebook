Ques 7 : How to pass lambda expressions as an argument in java 8?

Before java 8, we cannot pass a method as an argument.

Basic Example

public class Demo
{
  public static int getResult(int x, int y, Operation opt)
  {
      return opt.perform(x,y);
  }

  public static void main(String args[])
  {
      int sum = getResult(5,10, (x,y) -> x+y);
      int sub = getResult(5,10, (x,y) -> x-y);
  }
}

interface Operation
{
  public int perform(int x, int y);
}
